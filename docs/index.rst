GaiaXPy is a Python library to facilitate handling Gaia BP/RP spectra as distributed from the `Gaia Archive <https://gea.esac.esa.int/archive/>`_.

Package overview
================
.. list-table::
   :widths: auto
   :header-rows: 1

   * - Subpackage
     - Description
   * - Calibrator
     - Implementation of the calibrator tool.
   * - Cholesky
     - Subpackage that implements the Cholesky functionality (inverse covariance, etc.).
   * - Converter
     - Implementation of the converter tool.
   * - Core
     - Constants and internal generic functions.
   * - Generator
     - Subpackage to generate synthetic photometry
   * - Plotter
     - Subpackage to plot the output of some of the tools.
   * - Spectrum
     - Implementation of different types of spectra.
